2048-python
===========

Based on the popular game [2048](https://github.com/gabrielecirulli/2048) by Gabriele Cirulli, here is a Python version that uses TKinter. 

![screenshot](img/screenshot.png)

To start the game, run:
    
    $ python3 puzzle.py
